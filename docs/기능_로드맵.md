# 기능 로드맵 (앞으로 추가할 기능)

> 사용자가 불러준 "필요한 기능"을 모아두는 곳이에요. 정해지면 진행상태도 같이 표시해요.
> (✅ 완료 / 🔵 진행중 / ⬜ 예정 / ❓ 결정 필요)

최초 작성: 2026-06-23

---

## 1. 예약 시스템 (핵심)

- ⬜ **예약 기능** — 손님이 직접 예약할 수 있는 방법 마련
  - ❓ **결정 필요**: 예약 방식 — 우리 홈페이지에서 자체 예약 vs 네이버/외부 예약 연동
- ⬜ **예약금 자동 처리** — 예약할 때 예약금을 자동으로 받고 확인되면 자동 승인
- ⬜ **예약 취소 기능** — 손님이 예약을 취소할 수 있게

## 2. 자동 문자(SMS) 발송 시스템

- ⬜ **예약 안내/확인 문자 자동 발송**
  - 📌 **메일(이메일)은 필요 없음** — 문자(SMS)만!

## 3. 리뷰 페이지

- ⬜ **리뷰(후기) 페이지** — 손님이 후기를 남기고 볼 수 있는 페이지

---

## ❓ 먼저 정해야 할 것 (결정 대기)

1. **고객을 무엇으로 구분할지** — **아이디(회원가입)** 로 할지, **전화번호** 로 할지 결정 필요
   - 전화번호 방식: 가입 없이 간편, 문자 발송과도 잘 맞음
   - 아이디 방식: 회원 관리·리뷰 작성자 추적 등에 유리
2. **예약 방식** — 자체 예약 vs 외부(네이버 등) 연동

---

## 메모

- 위 기능들은 단순 HTML만으로는 어렵고, 예약/결제/문자 발송을 처리할 **서버(백엔드)** 가 필요할 수 있어요. (자세한 방식은 결정 후 정리)
