ROADMAP · 실행 계획

이 순서로 올립니다

임팩트 × 노력 기준. Quick Win(반나절)만 쳐도 체감이 확 뜨고, 단계 2까지 하면 "재설계 없이 한 급" 올라갑니다.

QUICK WIN 반나절 · 저위험 · 먼저
  • 01 🔴 모바일 햄버거 메뉴 실작동 구현 — 지금 먹통(가장 심각)
  • 02 브랜드 컬러 정본화 — 밝은 파랑 → 공식 3색(#043CB2·#622698·#B20E19). CSS 변수값만 바꿔 전 페이지 반영
  • 03 미정의 토큰 정의(버그) + Primary CTA 생기 — --amber/--green 정의, 버튼 그라디언트+글로우
  • 04 포커스 링 복원 + 소형 텍스트 대비 상향 — :focus-visible, --faint→--muted
단계 2 1~2일 · 임팩트 최대
  • 05 히어로 시네마틱화 — Ken Burns 줌 + 패럴랙스 + 진입 스태거 + 비네트/그레인 (CSS만, 영상 없이 시작)
  • 06 디스플레이 폰트 + 타입 스케일 토큰화 — 제목 전용 폰트(next/font self-host), 본문 Pretendard 유지
  • 07 다크 크래프트 폴리시 — 오로라 글로우 + 카드 그라디언트 보더/글로우 + 전역 그레인 1장
  • 08 홈 리뷰/평점 섹션 신설 — 별점 요약 + 후기 발췌(기존 /reviews 데이터 재사용)
단계 3 여력 시 · 완성도
  • 09 스무스 스크롤 + reveal 스태거 — 조작감 레벨업(콘텐츠 무변경)
  • 10 스티키 핀 연출 1곳 — "무대 위에 세웁니다" Apple식
  • 11 next/image 전환 + 폼 시맨틱화 — LCP/CLS, 선택칩 button화
  • 12 radius 3단계 통일 + 인라인 스타일 토큰 정리

⚠️ 범위 원칙: 위 전부 현재 구조·정체성을 그대로 두고 값·연출·컴포넌트 디테일만 올리는 작업입니다. 페이지 구성·콘텐츠·URL은 안 바뀝니다. 라이브에 반영하려면 각 항목을 코드에 적용 후 git push → 미리보기 확인 흐름으로 진행합니다.
