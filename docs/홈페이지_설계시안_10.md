# 판타스트릭 홈페이지 — 전체 설계 시안 10가지 (B2B + B2C)

> 작성 2026-06-18. 경쟁사 홈페이지 구조 리서치 → 우리 목적(브랜드 소개 · 컨설팅 · 콘텐츠/테마/장치 외주 · 기술·장치 판매 · 매장/테마 소개 · 예약)에 맞춘 설계 방향 10종.

## 홈페이지 목적 (확정)
- **B2C**: 브랜드 소개, 보유 매장(3곳)·테마(4종: 태초의 신부, 사자의 서, 락다운시티, 시간의 영속성) 소개, **예약**
- **B2B**: 컨설팅, 콘텐츠(테마)·장치 **외주 제작**, 보유 **기술·장치 소개 및 판매**, 제휴/문의

### 핵심 시사점
1. **업계는 예약을 거의 다 자체 호스팅**(php)으로 운영 → 우리도 `fantastrick.co.kr` 예약을 중심 동선으로.
2. **B2B/B2C를 한 사이트에 담는 검증된 2패턴**: (A) 키이스케이프식 **도메인/포털 분리**, (B) 셜록홈즈식 **단일 내비 병치**.
3. **외주 역량은 CONTENT(콘텐츠)·SPACE(공간)·TECH(장치) 3축**으로 구조화하면 설득력↑ (키이스케이프 검증).
4. **NN/g UX 원칙**: 'B2B/B2C' 같은 청중 라벨 분리는 라벨이 명확·배타적일 때만. 모호하면 **"무엇을 원하세요?" 니즈 기반**으로 가르는 게 안전.
5. 핌아트웍스처럼 **제작사 포트폴리오형**은 신뢰는 높지만 일반 손님 예약 동선이 약해짐 → 우리는 예약을 죽이면 안 됨.

## 2. 설계 시안 10가지

각 시안: 컨셉 / IA(메뉴) / 메인 섹션 순서 / 톤 / 장단점 / 추천도

### 시안 1. 입장 게이트 (Split-Gate) ★★★☆☆
- **컨셉**: 첫 화면에서 두 문 — "체험하러 왔어요(B2C) / 함께 만들래요(B2B)". TGC '게임 입장' 세계관과 찰떡.
- **IA**: 게이트 → B2C(테마/매장/예약/소식) · B2B(컨설팅/콘텐츠·장치/레퍼런스/문의)
- **메인 순서(B2C)**: 히어로 → 테마 그리드 → 매장 → 예약 → 소식 → 푸터
- **톤**: 다크 시네마틱 · 시안 네온
- **+** 타깃별 메시지 가장 명확  / **–** 클릭 1번 더(이탈), NN/g 경고
- 

### 시안 2. 본관 + 비즈니스관 (키이스케이프식 듀얼) ★★★★★
- **컨셉**: B2C가 메인, B2B는 별도 섹션/서브도메인. 업계 1위가 쓰는 정석.
- **IA**: 메인 = 소개/테마/매장/예약/소식/[비즈니스] · 비즈니스 = 포트폴리오/서비스(컨설팅·콘텐츠·장치)/기술·장치 판매/단체·대관
- **메인 순서**: 히어로 → 브랜드(11년·세계관) → 테마 → 매장 → 예약 → 소식 → 푸터(비즈니스 진입)
- **톤**: 다크 프리미엄
- **+** 검증·확장성 최고  / **–** 페이지 많아 제작량↑
- **장기적으로 가장 안전한 정석**

### 시안 3. 한 줄 병치 (셜록홈즈식 단일 내비) ★★★★☆
- **컨셉**: 한 내비에 B2C·B2B 모두. 예약하기 옆에 "제휴·외주 문의".
- **IA**: 소개 / 테마·매장 / 예약 / 컨설팅·외주 / 기술·장치 / 문의 (+KOR·ENG)
- **메인 순서**: 히어로 → 테마 → 매장 → 기술·장치 → B2B 배너 → 예약 → 푸터
- **톤**: 다크 · 깔끔
- **+** 단일 사이트로 단순·가벼움  / **–** 메뉴 길어짐, 타깃 혼재
- **빠른 시작에 현실적**

### 시안 4. 스크롤 내러티브 원페이지 ★★★★☆
- **컨셉**: 긴 한 페이지를 영화처럼 — 일상이 멈추는 순간(히어로)→세계관→테마→매장/예약→"우리가 만듭니다"(B2B)→보유 기술·장치→문의.
- **IA**: 앵커 내비(스크롤 이동) + 예약 버튼 고정
- **톤**: 시네마틱 풀스크린 · 패럴랙스
- **+** 몰입·스토리텔링 최강, 모바일 친화  / **–** 정보 많은 B2B엔 다소 얕음
- **브랜딩 임팩트 강력**

### 시안 5. 니즈 허브 (NN/g 권장) ★★★★☆
- **컨셉**: 청중 라벨 대신 "무엇을 원하세요?" — 예약 / 단체·대관 / 테마 제작 의뢰 / 장치 도입 / 컨설팅.
- **IA**: 허브 카드 5개 → 각 목적 페이지
- **메인 순서**: 히어로 → 니즈 카드 허브 → 대표 테마 → 레퍼런스 → 푸터
- **톤**: 미니멀·구조적
- **+** UX 정석, 혼동 최소  / **–** 브랜드 감성 약해질 수 있음

### 시안 6. 스튜디오 포트폴리오 (핌아트웍스식) ★★★☆☆
- **컨셉**: "우리는 만든다"를 전면. 제작 에이전시 이미지, B2B 우선·예약 보조.
- **IA**: ABOUT / WORKS(테마·프로젝트) / PROCESS / 기술·장치 / CONTACT (+예약)
- **메인 순서**: 히어로(작업물 릴) → 역량 3축 → 대표 프로젝트 → 매장/예약 → 문의
- **톤**: 갤러리·매거진, 큰 이미지
- **+** 외주 수주 신뢰↑  / **–** 일반 손님 예약 동선 약화
- **B2B 비중 크게 갈 때**

### 시안 7. TGC 게임센터 몰입형 ★★★☆☆
- **컨셉**: Tricky Game Center 세계관으로 사이트 자체를 게임처럼. Fanta·Tricky 캐릭터, 업적/DLC 모티프, 인터랙션.
- **IA**: 입장 → 테마(스테이지)/매장(거점)/예약/세계관·캐릭터/비즈니스(제작소)
- **톤**: 다크 게이미피케이션 · 네온 · 애니메이션
- **+** 차별화·바이럴·IP 강화  / **–** 제작비·시간↑, B2B 진지함과 톤 충돌 가능
- **개성 최강(리스크 있음)**

### 시안 8. 테마 카탈로그 중심 ★★★★☆
- **컨셉**: 방탈출 본질=테마. 메인이 곧 테마 그리드(필터: 매장·장르·난이도·인원·시간), 예약 직결.
- **IA**: 테마 / 매장 / 예약 / 소개 / 비즈니스
- **메인 순서**: 짧은 히어로 → 테마 필터 그리드 → 매장 지도 → 예약 → B2B 배너 → 푸터
- **톤**: 다크 · 카드 UI · 실용적
- **+** 예약 전환율↑, 목적 직빵  / **–** 브랜드 스토리·B2B 약함
- **B2C 전환 우선**

### 시안 9. 기술·장치 쇼케이스/커머스 ★★★☆☆
- **컨셉**: 보유 기술·장치 판매를 전면 카탈로그로. "이머시브 장치 솔루션" 제품 페이지·스펙·문의.
- **IA**: 솔루션(장치)/제작·컨설팅/레퍼런스/브랜드(매장·테마)/예약/문의
- **메인 순서**: 히어로(기술력) → 장치 카탈로그 → 적용 사례(우리 테마) → 컨설팅 → 예약 → 문의
- **톤**: 테크·프리미엄, 제품샷
- **+** 장치 판매·외주 리드↑  / **–** B2C 감성 약화
- **장치 사업 키울 때**

### 시안 10. 듀얼 히어로 반반 ★★★★☆
- **컨셉**: 첫 화면을 좌/우(모바일은 상/하)로 반 갈라 두 입구 동시 — 왼쪽 "예약·체험"(B2C), 오른쪽 "외주·컨설팅·장치"(B2B). 한 화면에서 둘 다.
- **IA**: 통합 내비 + 히어로에서 시각적 분기
- **메인 순서**: 듀얼 히어로 → (공통)브랜드 → 테마/매장 → 비즈니스 역량 → 예약·문의 → 푸터
- **톤**: 대칭 split · 다크
- **+** 분기+단일사이트 장점 결합, 클릭 손실 적음  / **–** 첫인상 다소 분산

| 우선순위 | 적합 시안 |
|---|---|

| **빠른 시작·단일** | 시안 3 (한 줄 병치), 시안 10 (듀얼 히어로) |

### 추천(단계 전략)
판타스트릭 현 단계(B2C 예약 회복 + B2B 외주 시드머니 확대 중)엔 **하이브리드**가 좋다:
> **시작**: 시안 4(스크롤 몰입 히어로) + 시안 3/10(단일 사이트 내비 병치) + 시안 8(테마 그리드)를 결합 → 한 사이트에서 몰입·예약·B2B 문의를 모두.
> **성장 후**: B2B가 커지면 시안 2처럼 **비즈니스 서브포털로 분리**.
