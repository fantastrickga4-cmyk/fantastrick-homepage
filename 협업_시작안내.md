# 판타스트릭 홈페이지 — 협업 시작 안내

안녕하세요! 판타스트릭 홈페이지 프로젝트에 합류해주셔서 감사합니다.
이 문서대로 따라 하시면 **내 컴퓨터로 프로젝트를 받아 → 작업 → 다시 올리기**까지 할 수 있어요.

---

## 0. 미리 준비할 것 (한 번만)

1. **GitHub 계정** — 없으면 https://github.com 에서 무료 가입
2. **저장소 초대 수락** — 관리자(fantastrick2026)가 보낸 초대 메일/알림에서 **Accept** 클릭
   - 이 저장소는 **비공개(Private)** 라서, 초대를 수락해야 접근돼요.
3. **Git 설치** — https://git-scm.com 에서 다운로드 후 설치
4. (선택) **Claude Code** — 본인 환경에 맞게 설치해두면 AI로 작업 가능

> 저장소 주소: **https://github.com/fantastrick2026/fantastrick-homepage**

---

## 1. 프로젝트 받기 (복제 / clone) — 한 번만

터미널(명령창)에서 원하는 폴더로 이동한 뒤:

```bash
git clone https://github.com/fantastrick2026/fantastrick-homepage.git
cd fantastrick-homepage
```

- 비공개 저장소라 **GitHub 로그인**을 물어볼 수 있어요. 브라우저/계정으로 로그인하면 됩니다.
- 복제가 끝나면 `fantastrick-homepage` 폴더에 모든 파일이 들어와요.

---

## 2. Claude Code로 작업하기

1. 복제한 `fantastrick-homepage` 폴더에서 **본인의 Claude Code**를 실행
2. Claude가 **제일 먼저 읽어야 할 약속/현황 문서**:
   - `CLAUDE.md` — 이 프로젝트에서 Claude가 지킬 규칙 (꼭 먼저!)
   - `README.md` — 지금까지 무엇을 했는지 변경 기록
   - `DIRECTORY_MAP.md` — 폴더/파일 지도

### 주요 파일 위치
| 무엇 | 경로 |
|---|---|
| 메인 페이지(B2C) 프로토타입 | `docs/하이브리드_설계.html` |
| 비즈니스(B2B) 페이지 | `docs/비즈니스.html` |
| 설계 시안 모음·미리보기 | `docs/시안미리보기.html`, `docs/홈페이지_설계시안_10.md` |
| 이미지(로고·포스터·지도 등) | `assets/images/` |

### 미리보기(화면 확인)
- 빌드 필요 없어요. **`docs/하이브리드_설계.html` 파일을 크롬으로 더블클릭**하면 바로 보여요.
- (선택) 화면 캡처 도구를 쓰려면 한 번만: `npm install` (Playwright 설치됨)

---

## 3. 작업 흐름 (매번)

### ① 작업 시작 전 — 최신으로 받기
```bash
git pull
```
> 다른 사람이 올린 변경을 먼저 받아와야 충돌이 안 나요. **작업 전 습관적으로!**

### ② 작업 (파일 수정)

### ③ 작업 후 — 저장하고 올리기
```bash
git add -A
git commit -m "무엇을 했는지 한국어로 간단히"
git push
```

---

## 4. 지켜주시면 좋은 약속

- **작업 전 `git pull`**, 작업 후 **`commit` + `push`** — 작게 자주.
- **커밋 메시지는 한국어로 간단하게** (예: "테마 카드 호버 효과 수정").
- 큰 변경을 했으면 **`README.md`에 한 줄 기록** (이 프로젝트 규칙).
- 가능하면 **서로 다른 파일/영역**을 작업 (같은 곳 동시 수정 시 충돌 가능).
- 충돌(merge conflict)이 나도 당황하지 마세요 — 관리자나 Claude에게 도움 요청하면 됩니다.

---

## 5. 자주 막히는 것

- **"clone이 안 돼요 / 권한 없음"** → 저장소 초대를 아직 수락 안 했을 수 있어요(0번 2단계).
- **push가 거부됨(rejected)** → 내 작업 전에 누가 먼저 올렸을 때예요. `git pull` 먼저 하고 다시 `git push`.
- **로그인 창이 안 뜸/실패** → Git 설치 시 함께 오는 "Git Credential Manager"로 GitHub 로그인하면 돼요.
